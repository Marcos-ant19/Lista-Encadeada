Lista Encadeada em C

Projeto desenvolvido em linguagem C com o objetivo de implementar uma lista encadeada dinâmica utilizando alocação de memória.

Sobre o projeto

Este projeto tem como objetivo praticar conceitos fundamentais de Estruturas de Dados, com foco na implementação de listas encadeadas dinâmicas em C.

A estrutura permite armazenar e manipular dados de forma flexível por meio do uso de ponteiros e alocação dinâmica de memória.

Conceitos aplicados
Ponteiros
Structs
Alocação dinâmica de memória
Encadeamento de nós
Modularização de código
Estruturas de dados lineares
Objetivo acadêmico

Reforçar o aprendizado sobre estruturas de dados dinâmicas e desenvolver maior domínio sobre manipulação de memória na linguagem C.
